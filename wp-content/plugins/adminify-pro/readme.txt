## WP Adminify - Powerhouse Toolkit for WordPress Dashboard ##
This plugin gives you access to the following feature:

1. [Admin Columns Customizer](https://wpadminify.com/admin-columns/): By default, WordPress provides a list of admin columns for different post types. Using this module, you can customize those admin columns and add or remove them when it's required.

2. [Post type & Media Folders](https://wpadminify.com/media-folders/): Folders module will help you quickly organize all of your Pages, Posts, and Media. With it, you can create unlimited folders to give your posts the best home they could ever want.


3. [Google Pagespeed Insights](https://wpadminify.com/google-pagespeed-insights/): Google Pagespeed Insights is a tool that helps you to find out issues for better page speed. WP Adminify integrated Google lighthouse inside this module, it will provide you an in-depth page optimization suggestions for a fast page.


4. [Disable Admin Notices](https://wpadminify.com/admin-notices/): Do you know what it's like when a plugin notifies you to upgrade to the paid version, collects your technical data & then shows a never-ending onslaught of notifications? Maybe it interrupts your work with WordPress? With the help of disable comments module, you can get rid of all notice in a second.


5. [Dashboard Menu Editor](https://wpadminify.com/menu-editor/): If you're looking to edit the admin menu, then you should try this Admin Menu Editor by WP Adminify. With just a few clicks, you can change user role permissions, edit default text, URL, and icon.


6. [WordPress Login Customizer](https://wpadminify.com/login-customizer/): Login Customizer by WP Adminify comes with many customization settings to personalize your WordPress login page. You can change the style, layout, typography and even some of its textual elements like label, placeholder or error messages too.


7. [WordPress Activity Logs](https://wpadminify.com/activity-logs/): If you want to track every event happening on your WordPress website, you need to use the WP Adminify plugin. It's like a hidden security camera that tracks what happens in the pages, posts, etc, and tells you exactly who did what.


8. [Disable Comments for WordPress](https://wpadminify.com/disable-comments/): Spammers always love the comment form. You can disable the comment form website URL field and disable link in the comment form, author URL too. With a single click, you can get rid of all comment form on your website.


9. [Notification Bar for WordPress](https://wpadminify.com/notification-bar/): Notification Bar by WP Adminify enables the admin to create notification bars in the top or bottom position for the full site or page or post. The notification bar features a text message, and a button (optional). it can be configured to show on desktop and mobile devices. You can use it as a cookie notice by defining the position at the bottom.


10. [Dashboard & Welcome Widget](https://wpadminify.com/dashboard-welcome-widgets/): Input text, icon, video, shortcode, RSS feed content inside your Dashboard widget. The welcome widget can be customized in the normal text editor or Elementor page builder.


11. [Header / Footer Script](https://wpadminify.com/header-footer-scripts/): You can just use some CSS and JS codes to customize your WordPress site without modifying any other plugin or theme files. It has options to enable snippets for specific devices, pages, posts, taxonomy pages.


12. [Quick Floating Menu](https://wpadminify.com/quick-menu/): With the help of floating dashboard quick menu, you can arrange your necessary pages here. No matter where you navigate in your dashboard, it will be always accessible.


13. [Menu Duplicator for WordPress](https://wpadminify.com/menu-duplicator/): Duplicating a full menu or specific menu item is super easy with WP Adminify. Within a single click, you can duplicate any menu or menu item. After duplicate just customizes it to your needs.


14. [Admin Pages For WordPress](https://wpadminify.com/admin-pages/): Admin page module can help you to create parent or sub-pages under any existing dashboard menu item. You can design your custom admin page using any page builder like Visual composer, Elementor, Oxygen, etc.


15. [WordPress Post Duplicator](https://wpadminify.com/post-duplicator/): Do you like to create a different page or post in the same layout but with different content? Arranging a page or post from sketch is painful and time-consuming. Here Post Duplicator by WP Adminify can help you. Within a single click, you will get a full clone of your main page or post or post type.


16. [WordPress Sidebar Generator](https://wpadminify.com/custom-sidebar/): Create unlimited WordPress sidebar. Input title, description for your sidebar and sort them anytime.


17. [Post Type order module](https://wpadminify.com/post-types-order/): It's a powerful module that helps you to sort or order any type of post type. Not only post type but also it works for media and taxonomy order too. You can prevent different user roles to order post types too.


18. [Server Information](https://wpadminify.com/server-information/): WP Adminify presents some necessary information about your server. The amazing feature is, you can check your WordPress website debug log here. Not only check but also you can refresh and clean your debug file. It will show you a lot of WordPress Constants for the wp-config.php file.
